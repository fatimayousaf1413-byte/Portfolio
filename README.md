# Personal Portfolio Website

This is a responsive personal portfolio website built as the first internship task using HTML, CSS, and JavaScript.

## 🚀 Features
- Responsive design for all devices  
- Clean and modern UI  
- Smooth layout and simple navigation  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

## 📁 Project Structure
- index.html → Main page  
- style.css → Styling  
- script.js → Interactivity  
- images/ → Assets and images  

## ▶️ How to Run
1. Clone or download the repository  
2. Open the folder  
3. Run `index.html` in your browser

This is my first internship project focused on building a professional portfolio website using frontend technologies.
